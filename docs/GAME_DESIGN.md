# PacBecca Game Design

## Pitch

PacBecca is a bright, fast, personal maze-chase game where Becca clears 10 themed levels while four named troublemakers try to box her in.

## Starting Scope

- Browser game.
- Keyboard controls.
- One original maze for the first version.
- Ten level configs that scale speed, color, and vulnerability timing.
- Four ghosts with named personalities.
- Replaceable Becca head image.

## Level Set

1. Porch Practice
2. Glitter Hall
3. Left Turn Energy
4. Snack Dash
5. Orbit Hour
6. The Group Chat
7. Corner Confidence
8. Afterparty Loop
9. Close Call Club
10. Becca Mode

## Ghosts

- Riff: direct pursuer.
- Pippa: ahead-of-player ambusher.
- Orbit: flanker that uses Riff's position.
- Moxie: drifter who backs off when close.

## Asset Direction

The current Becca head is a friendly placeholder SVG. The real version should use a friend-approved photo, cropped to the head, ideally transparent, square, and at least 256 by 256 pixels.
